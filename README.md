# 👋 안녕하세요, 차희선입니다

🚀 Android 개발자 | Kotlin & Jetpack Compose  
🎓 네이버 부스트캠프 웹·모바일 10기 (Android)

복잡한 데이터를 직관적인 사용자 경험으로 바꾸는 개발을 좋아합니다.  
구조적인 설계와 유지보수 가능한 아키텍처를 고민하며 성장하고 있습니다.

---

## 🙋‍♂️ 소개

- 📱 Kotlin 기반 Android 앱 개발 (Jetpack Compose)
- 🏗 MVVM / MVI 아키텍처 설계에 관심이 많습니다
- 🔍 데이터 구조와 시스템 관점에서 문제를 해결하는 것을 즐깁니다
- 🎨 Figma 기반 협업과 UI/UX 개선 경험이 있습니다
- 🌱 Flutter를 활용한 크로스플랫폼 개발도 학습 중입니다

## 📌 주요 프로젝트

### 📚 책결 (Chaek-Gyeol)
소설 속 복잡한 인물 관계를 시각화하여 기록하는 독서 보조 서비스

- Jetpack Compose 기반 캔버스 인터랙션 UI 구현
- 화면 특성에 따라 **MVI / MVVM 아키텍처를 선택적으로 적용**
- 인물 관계를 **방향 그래프(Directed Graph)** 구조로 모델링
- 확대/축소 UX 개선 및 정보 가독성 최적화

---

### 🌡 온습도 모니터링 시스템 구축 및 고도화 (실무 프로젝트)
센서 기반 환경 데이터를 수집하고 시각화하며, 기준치 이탈 시 알림을 제공하는 서비스

- **1분 단위 시계열 데이터 시각화** 및 인터랙티브 툴팁 UI 구현  
- Android OS 정책으로 인한 백그라운드 수집 중단 문제를  
  **Launcher Mode(런처 앱) 도입으로 해결**
- AWS DynamoDB 기반 실시간 데이터 분석 로직 설계
- Firebase Cloud Messaging(FCM)을 활용한 이상 수치 알림 시스템 구축
- Date/Range Picker, 공통 Dialog 등 재사용 가능한 UI 컴포넌트 제작
